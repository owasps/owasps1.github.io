1.2 ls命令说明

>ls 是list的缩写 相当于DOS下的吃的dir

>以 . 文件名 形式的为隐藏文件 可用 ls -a [查询所有]查询到

> . 为当前目录 .. 为上级目录 [可与cd一起使用]

2.切换目录

2.1 cd (特别简单)

“~” 表示家目录

 2.2相对路径与绝对路径

>相对 之一级一级的路径 不能直接跳转

>绝对 可是实现文件夹之间的直接跳转

3.创建和删除操作

3.1 touch

创建文件 不存在创建文件 若存在 替换文件日期

3.2 mkdir

创建目录

若与 -p 连用则可以递归创建

mkdir -p a/b/c/d //递归创建 a中有b —> b中有c ……

Linux中文件与目录不能重名(windows 中则不是)，重名报错

3.3 rm 删除文件或目录（直接删除 = windows中 shift + Delete）

终端命令输入时 直接删除选择的文件，不会再垃圾桶中有临时文件，无法恢复

rm 删除文件

\1. -r rm -r 文件名 文件或目录

\2. -f rm -f 文件名 强制删除文件或目录(没有文件),而不会报错

4.拷贝和移动文件

4.1 tree

>以树状形式显示目录结构（文件目录都显示）

>若和-d 连用 则只显示目录不显示文件

4.2cp

cp复制命令 相当于DOS下的 copy

“ . “代表当前文件夹

cp ~/文档/123.txt ./123.txt = cp ~/文档/123.txt .

在桌面位置下 把音乐文件夹的文件复制到桌面(当前文件夹)下

在音乐位置下 把音乐(当前)文件夹的文件复制到桌面文件夹下

> -i cp -i ~/文档/123.txt ~/桌面

将文档文件cp到桌面文件下 若桌面下有相同文件 则-i命令提示是否覆盖桌面文件 输入 y（覆盖） n（不覆盖 保持不变）

-r 编辑目录是必须用的

> -r cp -r 2 test

将桌面目录2 复制一份 起别名为test

4.3 mv

移动命令 直接移动 不需要多余的语句 移动目录是也不用加 -r

多级目录直接调用

> -i mv -i a.txt b.txt

将文档文件mv到桌面文件下 若桌面下有相同文件 则 -i 命令提示是否覆盖桌面文件 输入 y（覆盖） n（不覆盖 保持不变）

5.查看文件内容

5.1 cat

cat 123.txt

cat -b 123.txt 显示每行编号（不包括空行）

cat -n 123.txt 显示每行编号（包括空行）

5.2 more

more 123.txt

5.3 grep

grep 123 123.txt

-n 查询数据的行号

grep -n 123 3.txt

-v 查询不为123的数据行

grep -v 123 3.txt

-i 忽略大小写

查询时忽略大小写 （查询一句话时 加“”）

grep -i “Hello python” 3.txt

常用两种模式查找

^a (匹配以a为头的)

grep ^f 123.txt

k$ (匹配以k为尾的)

grep a$ 123.txt

06.其他

6.1 echo(常于重定向联合使用)

echo hello

6.2 重定向 > 和 >>

可把终端输入的内容输入到指定文件中

> 覆盖文件原由内容

echo hello > a

重定向个文件a 存入 hello 数据（文件a中数据被 覆盖 ）

>> 添加内容

echo python >> a

重定向个文件a 存入 python 数据（在文件a中添加数据 不覆盖 ）

6.2 管道 |

ls -lha ~ | more

第一个命令的结果 传递给 第二个命令 第二个命令处理第一个命令输出

ls -lha ~ | grep x
